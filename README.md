# Mozaïk value widgets

[![NPM version][npm-image]][npm-url]

## Value Widget Configuration

### using `config.js`

```javascript
{
  //…
  value: {
    
  }
}
```

### Parameters

key       | required | description
----------|----------|-------------------------------------------------------------------
`url`     | yes      | *The URL to fetch the value*
`prefix`  | no       | *Prefix of the value*
`postfix` | no       | *Postfix of the value*
